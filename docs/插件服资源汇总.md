# 插件服资源汇总

## 插件下载平台

为了确保安全，请尽量从官方或知名的社区下载插件，**不要**轻易相信未知的整合包或第三方下载站 

- **[SpigotMC](https://www.spigotmc.org/resources/)**: 全球最大的插件社区，资源最丰富 
- **[Hangar](https://hangar.papermc.io/)**: PaperMC 官方推出的现代化插件平台，审核严格，质量高 
- **[Modrinth (Plugins)](https://modrinth.com/plugins)**: 新兴的平台，界面友好，下载速度快 
- **[BukkitDev](https://dev.bukkit.org/)**: 历史悠久的插件站 

## 常用插件推荐

### 基础/管理
- **[EssentialsX](https://essentialsx.net/)**: 提供几百个基础指令 (如 `/home`, `/tpa`, `/warp`, `/spawn`)，几乎是服务器标配 
- **[LuckPerms](https://luckperms.net/)**: 目前最好用的权限管理插件，拥有 Web 编辑器  
- **[Vault](https://www.spigotmc.org/resources/vault.34315/)**: 经济系统的前置插件，大多数经济插件都需要它 

### 领地/保护
- **[Residence](https://www.spigotmc.org/resources/residence.11480/)**: 经典的领地插件，玩家通过圈地保护建筑 
- **[GriefPrevention](https://www.spigotmc.org/resources/griefprevention.1884/)**: 通过金铲子圈地，更加原版化 
- **[CoreProtect](https://www.spigotmc.org/resources/coreprotect.8631/)**: 记录方块破坏、放置、容器交互等，用于回档和查询破坏者 

### 娱乐/功能
- **[Multiverse-Core](https://www.spigotmc.org/resources/multiverse-core.390/)**: 多世界管理，允许服务器拥有多个世界 
- **[SkinsRestorer](https://www.spigotmc.org/resources/skinsrestorer.2124/)**: 允许离线(盗版)服务器显示正版皮肤

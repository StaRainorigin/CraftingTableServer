# 开启一个插件服务器

插件服务器（如 Paper, Spigot）允许你安装各种插件来增强服务器功能（如领地、经济、登录系统等），同时保持客户端无需安装任何额外内容即可进入 

## 1. 下载服务端核心

推荐使用 **Paper**，它的性能优于原版和 Spigot，且兼容绝大多数 Spigot 插件 

- **PaperMC**: [Paper Downloads](https://papermc.io/downloads/paper)
- **Spigot**: [Get Bukkit](https://getbukkit.org/download/spigot) (需自行构建，不推荐新手)

## 2. 启动服务器

启动步骤与 [快速开始](快速开始.md) 中的原版服务器基本一致 

1. 将下载的 `.jar` 文件（如 `paper-1.20.4-380.jar`）放入空文件夹 
2. 为了方便，建议将其重命名为 `server.jar` 
3. 创建并运行启动脚本（`run.bat` 或 `run.sh`），内容参考 [快速开始](快速开始.md) 
4. 同意 `eula.txt` 

## 3. 安装插件

服务器成功启动一次后，你会发现根目录下多了一个 `plugins` 文件夹 

1. **下载插件**: 从可信源下载插件 `.jar` 文件 
2. **安装**: 将插件放入 `plugins` 文件夹 
3. **重启**: 在控制台输入 `stop` 关闭服务器，然后重新启动 
   - 或者在控制台输入 `restart` (不推荐使用 `/reload`) 

服务器启动时会加载插件，你可以在控制台看到插件加载的日志

请参考 **[插件服资源汇总](插件服资源汇总.md)**

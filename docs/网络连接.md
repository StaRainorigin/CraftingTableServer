# 网络连接

要让你的朋友进入你的服务器，你需要解决网络连接问题 根据你们的网络环境，有以下几种方式：

## 局域网 (LAN)

如果你的朋友和你连接的是**同一个 WiFi** 或路由器，或者你们在同一个宿舍/家庭网络下：

1. **获取你的局域网 IP**:
   - Windows: 打开 CMD，输入 `ipconfig`，找到 `IPv4 地址` (通常是 `192.168.x.x` 或 `10.x.x.x`) 
   - Linux/Mac: 终端输入 `ifconfig` 或 `ip a` 
2. **连接**:
   - 你的朋友在 "多人游戏" -> "添加服务器" -> "服务器地址" 中输入你的 **局域网 IP** 即可 

## 虚拟局域网 (Virtual LAN)

如果你们不在同一个地方，但不想折腾复杂的网络设置，可以使用虚拟局域网软件将你们模拟到同一个局域网中 

常用软件：
- **Radmin VPN** (Windows, 简单易用)
- **ZeroTier** (全平台, 稍微复杂但稳定)
- **Hamachi** (老牌软件)

**使用方法**:
1. 房主和玩家都安装并运行同一个软件 
2. 房主创建一个网络，玩家加入该网络 
3. 房主开启 Minecraft 服务器 
4. 玩家使用软件中显示的房主 **虚拟 IP** 进行连接 

## 内网穿透 (FRP / Tunnel)

如果你希望公开你的服务器，或者虚拟局域网效果不好，可以使用内网穿透服务 这通常需要使用第三方服务商或自己有一台公网服务器 

常用服务 (国内):
- **SakuraFrp**: 樱花映射，免费且好用，有详细教程 
- **OpenFrp**
- **Ngrok**

**原理**:
你运行一个客户端软件，将你本地的 `25565` 端口映射到服务商的公网服务器上 服务商会给你一个 `公网IP:端口` 或 `域名:端口`，你的朋友使用这个地址连接 

## 端口转发 (Port Forwarding)

这是最直接、性能最好的方式，但要求最高：
1. 你必须拥有**公网 IP** (光猫桥接/拨号，通常需要联系运营商申请) 
2. 你有路由器的管理员权限 

**设置步骤**:
1. 登录路由器后台 (通常是 `192.168.1.1` 或 `192.168.0.1`) 
2. 找到 "端口转发" / "虚拟服务器" / "NAT 设置" 
3. 添加规则：
   - 内部 IP: 你电脑的局域网 IP (如 `192.168.1.100`) 
   - 内部端口: `25565` 
   - 外部端口: `25565` (或者其他你喜欢的端口) 
   - 协议: TCP + UDP 
4. 保存生效 
5. 你的朋友使用你的 **公网 IP** 进行连接 (百度搜索 "IP" 即可查看) 

> **注意**: 公网 IP 可能会变动 (动态 IP)，建议配合 DDNS (动态域名解析) 使用

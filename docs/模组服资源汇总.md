# 模组服资源汇总

## 模组下载平台

- **[CurseForge](https://www.curseforge.com/minecraft/mc-mods)**: 老牌且最大的模组社区，资源最全 
- **[Modrinth](https://modrinth.com/mods)**: 开源、界面友好、无广告的现代化平台，越来越多的开发者选择在这里发布模组 

> **注意**: 许多模组在两个平台都有发布，建议优先查看 Modrinth 

## 常用服务端模组

### 性能优化
服务端优化模组能显著降低服务器延迟(TPS)和内存占用 

- **Fabric**:
  - **Lithium (锂)**: 通用优化，大幅提升物理、AI 等性能 
  - **Phosphor (磷) / Starlight**: 光照引擎优化 
  - **Krypton (氪)**: 网络栈优化 
- **Forge**:
  - **Embeddium / Rubidium**: 渲染优化(对服务端也有微小帮助，主要是客户端) 
  - **FerriteCore**: 内存占用优化 
  - **ModernFix**: 综合修复与优化 

### 管理工具
- **FTB Chunks**: 允许玩家按 `M` 键打开地图并认领区块(圈地)，防止破坏 
- **FTB Essentials**: 提供 `/home`, `/tpa`, `/spawn` 等基础指令 
- **Simple Voice Chat**: 简单语音聊天，让玩家在游戏中可以语音交流(需客户端也安装)

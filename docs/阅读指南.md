# 阅读指南

欢迎来到 CraftingTableServer 文档 本文档旨在帮助新手快速搭建自己的 Minecraft 服务器 

## 阅读顺序建议

### 1. 准备工作
在开始之前，请务必阅读 **[必要环境](必要环境.md)** 
Minecraft 服务器需要 Java 环境才能运行，这是最基础也是最容易出错的一步 

### 2. 第一次开服
如果你是第一次开服，建议先从 **[快速开始](快速开始.md)** 阅读 
这一章节会教你开启一个最简单的**原版服务器** 即使你的目标是插件服或模组服，也建议先走一遍原版流程，熟悉基本操作 

### 3. 让朋友加入
服务器开启后，你需要解决网络问题才能让朋友连接 
请阅读 **[网络连接](网络连接.md)**，根据你们的实际情况（局域网、异地）选择合适的方案 

### 4. 进阶：插件与模组
如果你不满足于原版生存，可以尝试：
- **[开启一个插件服](开启一个插件服.md)**: 适合纯净生存、小游戏服务器 
- **[开启一个模组服](开启一个模组服.md)**: 适合想玩各种 Mod 的玩家 

### 5. 深入了解
想要成为更专业的服主，建议阅读 **[概念介绍](概念介绍.md)**，了解服务端的核心文件结构、权限管理等知识

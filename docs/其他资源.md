# 其他资源

## 服务端管理面板

如果你觉得用命令行(黑框框)管理服务器太麻烦，或者想在网页上随时随地管理服务器，可以使用服务端管理面板 

### MCSManager
[MCSManager](https://mcsmanager.com/) 是一款开源、免费、支持中文的分布式管理面板 
- **优点**: 界面友好，支持 Docker，支持多节点管理，国内社区活跃 
- **适用**: 个人服主，小型商业服 

### Pterodactyl (翼龙面板)
[Pterodactyl](https://pterodactyl.io/) 是国外最主流的开源面板 
- **优点**: 极其稳定，生态丰富，Docker 隔离性好 
- **缺点**: 安装较复杂，主要面向商业托管商 

## 启动器

为了测试服务器，你也需要一个好用的 Minecraft 启动器 

- **[PCL2 (Plain Craft Launcher 2)](https://github.com/Hex-Dragon/PCL2)**: 
  - Windows 平台首选，界面华丽，功能强大，支持自动下载游戏、Mod、整合包 
- **[HMCL (Hello Minecraft! Launcher)](https://github.com/HMCL-dev/HMCL)**: 
  - 跨平台(Windows/Mac/Linux)，功能极其强大，支持自动安装 
- **[Prism Launcher](https://prismlauncher.org/)**: 
  - 开源、简洁、专注于多实例管理，适合硬核玩家 

## 实用工具网站

- **[MCalculator](https://mcal.zmh.me/)**: 
  - Minecraft 机制计算器，包含附魔、摔落伤害、红石等计算 
- **[Chunk Base](https://www.chunkbase.com/)**: 
  - 种子地图查询工具，可以查看生物群系、史莱姆区块、村庄、要塞位置等 
- **[Minecraft Wiki (中文)](https://zh.minecraft.wiki/)**: 
  - 遇到不懂的游戏机制，查 Wiki 永远是最好的选择
